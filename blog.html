Question-1: What are the different ways to select an element in the DOM?

Answer-1:
document.getElementById("id")-Select the ID
document.getElementsByClassName("class")-Selects elements by class
document.getElementsByTagName("tag") -Selects elements by tag
document.querySelector("selector")-Selects the first mapping element
document.querySelectorAll("selector")-Selects all mapping elements

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2:
innerHTML - Returns or sets the HTML content inside an element.Gets/sets HTML + text.
innerText - Returns or sets only the visible text of an element,ignoring hidden content.It dose not parse HTML
textContent - Retrieves or sets all text,including hidden content,without formatting .It is faster than innerText as it dose not consider CSS styling

Question-3: What is event delegation in the DOM?

Answer-3:Event delegation is a technique where you add an event listener to a parent element instead of multiple child elements.The event bubbles up from the child to the parent,allowing the parent to handle it.

Question-4: What is event bubbling in the DOM?

Answer-4:Event bubbleing is when an event starts from the target element and moves through its parent elements in the DOM
document.getElementById("child").addEventListener("click",()=>{
console.log("Child Clicked");
});
document.getElementById("parent").addEventListener("click",()=>{
console.log("Parent Clicked");
});
Output for click the child
Child Clicked
Parent Clicked
